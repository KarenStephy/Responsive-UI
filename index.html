About

Project description and contact

Book & Notes Vault helps students catalog books, lecture notes, research materials, and study resources in one accessible place.

  • Semantic layout with responsive cards on mobile and a table on desktop.
  • Regex validation and regex search with safe compilation and highlighting.
  • Dashboard statistics, reading targets, local persistence, and JSON import/export.
